## 📌 Key Concepts This Week

- **Binary (base-2)** uses only two digits: 0 and 1
- **Decimal (base-10)** uses ten digits: 0 through 9
- **Place values** in binary are powers of 2: 1, 2, 4, 8, 16, 32, 64, 128, …
- **Binary → Decimal:** Multiply each bit by its place value and add them up
- **Decimal → Binary:** Divide by 2 repeatedly, collect remainders, read bottom to top
- **Python functions:** `bin()` converts decimal to binary, `int(str, 2)` converts binary to decimal
- **Finger binary:** You can count 0–31 on one hand using binary!
