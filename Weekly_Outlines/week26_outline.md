## 📌 Key Concepts This Week

- **AND (`&`)** — returns 1 only when both bits are 1; like two switches in series
- **OR (`|`)** — returns 1 when either bit is 1; like two switches in parallel
- **XOR (`^`)** — returns 1 when the bits are different; used in encryption
- **NOT (`~`)** — flips all bits; in Python, `~x` equals `-(x + 1)`
- **Left Shift (`<<`)** — shifts bits left; equivalent to multiplying by 2ⁿ
- **Right Shift (`>>`)** — shifts bits right; equivalent to integer division by 2ⁿ
