ü§î Logical Errors: The Sneaky Bugs

üßÆ Math Test Analogy

Imagine you're solving 5 + 3, but you write down 6 instead of 8. Your handwriting is perfect, you followed all the rules, but the answer is wrong. That's a logical error!

üé≠Common Logical Errors

1. Swapped Even/Odd Logic:

num = 7
if num % 2 == 0:
    print(num, "is odd.")   # ‚ùå Logic is backwards!
else:
    print(num, "is even.")
Output: 7 is even. (WRONG!)

‚úÖ Fix it:

num = 7
if num % 2 == 0:
    print(num, "is even.")  # ‚úÖ Correct logic
else:
    print(num, "is odd.")
Output: 7 is odd. (CORRECT!)

üöó Road Trip Analogy

üí•Common Runtime Errors

1. NameError - Using undefined variables:

2. ValueError - Wrong type of value:

3. ZeroDivisionError - Dividing by zero:

ZeroDivisionError: division by zero

4. TypeError - Wrong operation for the type:

print(5 + "hello")  # ‚ùå Can't add number and string!

5. IndexError - Index out of range:

print(nums[5])  # ‚ùå Only indices 0, 1, 2 exist!
IndexError: list index out of range

üõ°Ô∏è Exception Handling: Your Safety Net

üé™ Trapeze Artist Analogy

A trapeze artist has a safety net below. If they fall (error happens), the net catches them instead of them hitting the ground. That's what try-except does for your code!

üéØThe Try-Except Structure

TRY Block: "Let me try this risky code..." 
try:
    x = int(input("Enter a number: "))
    result = 100 / x
    print("Result:", result)
EXCEPT Block: "Oops! If something goes wrong, do this instead..." 
except ZeroDivisionError:
    print("Can't divide by zero!")
except ValueError:
    print("That's not a valid number!")
ELSE Block (Optional): "If nothing went wrong, also do this..." 
else:
    print("Success! Thanks for the valid input.")
FINALLY Block (Optional): "No matter what happens, do this..." 
finally:
    print("Thank you for using this program!")
üí° Remember:
  • try: Required - code that might fail
  • except: Required - what to do if it fails
  • else: Optional - runs if try succeeds
  • finally: Optional - always runs, no matter what
