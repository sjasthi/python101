‚ú® Optional Arguments (Default Values)

What are Optional Arguments?

Optional arguments have default values in the function definition. You can skip them when calling the function!

Think of it like:

Ordering pizza: "I'll have a large pepperoni" (size and topping specified) vs "I'll have a pizza" (uses default size and topping) üçï

‚ö†Ô∏è Important Rule:

Parameters with defaults MUST come AFTER parameters without defaults!

# ‚úÖ Correct

    pass

    pass

  • You can override defaults by providing a value
  • Required parameters must come before optional ones in definitions
