Why Indentation Matters in Python

  • Python uses indentation to define code blocks - unlike other languages that use curly braces { }
  • Standard practice: Use 4 spaces for each level of indentation
  • Be consistent: Don't mix tabs and spaces
  • Common mistake: Forgetting to indent code inside if, else, elif blocks

Key Points:

  • All code inside the if block must be indented (4 spaces)
  • Both print statements are part of the if block

Example 3: IF-ELIF-ELSE Statement
‚ùå Incorrect Indentation

‚úÖ Correct Indentation

Key Points:

  • elif (else if) is at the same level as if
  • Each condition block (if, elif, else) must have indented code
  • You can have multiple elif statements
