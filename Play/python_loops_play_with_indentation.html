Why Indentation Matters in Loops

  • Loop body must be indented: All code inside a loop must be indented by 4 spaces
  • Nested loops need more indentation: Each level of nesting adds 4 more spaces
  • Be consistent: Use the same indentation throughout your code
  • Common mistakes: Forgetting to indent loop body, or mixing indentation levels
Example 1: Simple WHILE Loop

Key Points:

  • Code inside the while loop must be indented (4 spaces)
  • Both print and count statements are part of the loop body
  • The final print statement is NOT indented, so it runs after the loop
  • The loop continues as long as the condition is True
Example 2: Simple FOR Loop with range()
‚ùå Incorrect Indentation

‚úèÔ∏è Fix It Here:
‚úÖ Correct Indentation
for i in range(5): print(i) print(i * 2) print("Done with loop")

Key Points:

  • For loops require indentation for the loop body
  • range(5) generates numbers 0, 1, 2, 3, 4
  • Both print statements inside the loop are indented
  • The final print is NOT indented, so it runs once after the loop completes
Example 3: FOR Loop with IF Condition
‚ùå Incorrect Indentation
for num in range(1, 6): print(num) if num % 2 == 0: print("Even number") else: print("Odd number") print("All numbers checked")
‚úèÔ∏è Fix It Here:
‚úÖ Correct Indentation
for num in range(1, 6): print(num) if num % 2 == 0: print("Even number") else: print("Odd number") print("All numbers checked")

Key Points:

  • The if-else is INSIDE the for loop, so it needs 4 spaces
  • Code inside the if and else blocks needs 8 spaces (double indentation)
  • First level (for loop): 4 spaces
  • Second level (if/else inside loop): 8 spaces
Example 4: Nested WHILE Loops (Loop inside Loop)
‚ùå Incorrect Indentation
row = 1 while row <= 3: print("Row:", row) col = 1 while col <= 2: print(" Column:", col) col = col + 1 row = row + 1 print("Pattern complete")
‚úèÔ∏è Fix It Here:
‚úÖ Correct Indentation
row = 1 while row <= 3: print("Row:", row) col = 1 while col <= 2: print(" Column:", col) col = col + 1 row = row + 1 print("Pattern complete")

Key Points:

  • Nested loops require additional indentation levels
  • Outer while loop body: 4 spaces
  • Inner while loop body: 8 spaces
  • The inner loop completes fully for each iteration of the outer loop
  • Watch carefully where each loop ends!
Example 5: Nested FOR Loops (Multiplication Table)
‚ùå Incorrect Indentation

‚úèÔ∏è Fix It Here:
‚úÖ Correct Indentation
for i in range(1, 4): print("Table for", i) for j in range(1, 4): result = i * j print(i, "x", j, "=", result) print("---") print("Tables complete")

Key Points:

  • Nested for loops are common for patterns and tables
  • Outer loop (i): 4 spaces for its body
  • Inner loop (j): 8 spaces for its body
  • The "---" separator is at 4 spaces (part of outer loop, not inner)
  • "Tables complete" is at 0 spaces (runs after all loops finish)
