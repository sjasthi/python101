What is a Mixed List?

A mixed list (also called a heterogeneous list) is a list that contains different data types all in one list. Python is super flexible — your list can hold integers, strings, floats, booleans, and more, all at the same time!

💡 Backpack analogy: Think of a mixed list like your school backpack — it holds your textbook (string), pencil count (int), whether you did homework (bool), and your GPA (float) all at once!
📋

Homogeneous vs Heterogeneous

Same Type (Homogeneous)

All items are the same data type:

Python
# All integers
nums  = [3, 4, 5, 3]

# All strings
names = ["python", "java", "javascript"]

# All characters
chars = ['a', 'z', 'x']

Different Types (Heterogeneous / Mixed)

Items can be different data types — this is a mixed list:

Python
# A student record with mixed types
student = ['Srikar', 15, True, 3.94]
#           str      int   bool   float
IndexValueData Type
student[0]'Srikar'str
student[1]15int

🔍

Accessing Mixed List Items

Access items using their index (position), starting from 0:

Python
student = ['Srikar', 15, True, 3.94]

print(student[0])   # Output: Srikar
print(student[1])   # Output: 15
print(student[2])   # Output: True
print(student[3])   # Output: 3.94
print(student[-1])  # Output: 3.94 (last item!)
⚡ Remember: -1 is always a shortcut for the last item in any list!
✅

What Can a Mixed List Hold?

Python
# A mixed list can hold ANY of these:
my_list = [
    "hello",           # str  - text
    42,                # int  - whole number
    3.14,              # float- decimal number
    True,              # bool - True or False
    [1, 2, 3]           # list - even another list!
]

Data type badges: str int float bool list

What is a Nested List?

A nested list is a list that contains other lists as elements. Think of it like folders on your computer — a folder can contain sub-folders inside it!

🗂️ Folder analogy: Just like a folder can have sub-folders, a list can have other lists inside it!
🎒

A Student Record with Nested Lists

What if we want to store a student's courses and grades inside the same student list?

Python
student_1 = ["abe", 15, True, 3.9, "[email protected]",

Accessing Nested List Items

Use two sets of square brackets to drill into a nested list:

Python
student_1 = ["abe", 15, True, 3.9, "[email protected]",
             ['physics', 'maths', 'English'],
             [89, 95, 76]]

# Get the whole courses list
print(student_1[5])        # ['physics', 'maths', 'English']

# Get just the last course
print(student_1[5][-1])     # 'English'

# Get the first grade
print(student_1[6][0])      # 89

# Get the second course
print(student_1[5][1])      # 'maths'
🎯 Formula: list[outer_index][inner_index] — first pick WHICH inner list, then pick WHICH item inside it.
📖

Bookcase Example

Imagine a bookcase: some slots have a single book, others have a whole book series (a list)!

Python
book_case = [
    'python',                                                # index 0
    'maths',                                                 # index 1
    ["Hunger Games 1", "Hunger Games 2", "Hunger Games 3"], # index 2
    "science",                                               # index 3
    ["Harry Potter 1", "Harry Potter 2", "Harry Potter 3"]  # index 4
]

# Get the Hunger Games series
print(book_case[2])        # ['Hunger Games 1', 'Hunger Games 2', 'Hunger Games 3']

# Get just "Hunger Games 2"
print(book_case[2][1])     # 'Hunger Games 2'

# Get "Harry Potter 3" (last book)
print(book_case[4][-1])    # 'Harry Potter 3'
🧅

Deeply Nested Lists

Lists can be nested many levels deep! Each extra [index] peels one layer off, like an onion 🧅

Python
nested_x = ['a', ['bb', ['cxt', ['dy', 'xy'], 'ee', 'ff'], 'g', 'h']]

# Peel the layers one by one:
print(nested_x[1])             # ['bb', ['cxt', ['dy','xy'],'ee','ff'], 'g', 'h']
print(nested_x[1][1])          # ['cxt', ['dy', 'xy'], 'ee', 'ff']
print(nested_x[1][1][1])       # ['dy', 'xy']
print(nested_x[1][1][1][1])    # 'xy'
🧅 Each [index] goes one level deeper into the nested structure!

What is a 2D List?

A 2D list is a nested list where every inner list has the same number of elements, forming a neat grid — like a table, spreadsheet, or chessboard!

🏟️ Theater seat analogy: Think of a 2D list like movie theater seats — you always say Row first, then Seat (column). Same with 2D lists: list[row][col]!
🏗️

Accessing 2D List Elements

Use two indexes: list[row][col]

Python
scores = [[10, 25, 14],
          [46, 65, 72],
          [37, 98, 99]]

print(scores[0][0])  # → 10  (Row 0, Col 0)
print(scores[1][2])  # → 72  (Row 1, Col 2)
print(scores[2][1])  # → 98  (Row 2, Col 1)
print(scores[0][-1]) # → 14  (Row 0, last col)

Measuring a 2D List

Python
scores = [[10, 25, 14, 73],
          [46, 65, 72, 53],
          [37, 98, 99, 15]]

print(len(scores))      # → 3  (number of rows)
print(len(scores[0]))   # → 4  (number of columns)
# Total elements = 3 × 4 = 12
📏 len(list) = number of rows  ·  len(list[0]) = number of columns
🔴

Python – Row-Major Traversal
scores = [[10, 25, 14, 73],
          [46, 65, 72, 53],
          [37, 98, 99, 15]]

# Row by row (outer loop = rows, inner loop = columns)
for row in scores:
    for item in row:
        print(item, end=", ")

# Output: 10, 25, 14, 73, 46, 65, 72, 53, 37, 98, 99, 15,

Python – Column-Major Traversal
scores = [[10, 25, 14, 73],
          [46, 65, 72, 53],
          [37, 98, 99, 15]]

num_rows = len(scores)       # 3
num_cols = len(scores[0])    # 4

# Column by column (outer loop = columns, inner loop = rows)
for col in range(num_cols):
    for row in range(num_rows):
        print(scores[row][col], end=", ")

# Output: 10, 46, 37, 25, 65, 98, 14, 72, 99, 73, 53, 15,
💡 Key difference: In row-major the row loop is outer. In column-major the column loop is outer!
