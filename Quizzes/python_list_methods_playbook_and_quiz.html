üéØ What is a List?

A list is like a container that can hold multiple items in order. Think of it like a shopping cart üõí where you can put different items, or a playlist üéµ where you arrange your favorite songs in a specific order.

# Examples of Lists
fruits = ["apple", "banana", "cherry"]
numbers = [10, 20, 30, 40, 50]
mixed = [1, "hello", 3.14, True]
empty_list = []
üí° Key Points:
  • Lists are created using square brackets [ ]
  • Items are separated by commas
  • Lists can hold any type of data: numbers, strings, even other lists!
  • Items in a list maintain their order
  • Python starts counting from 0 (first item is at position 0)

üîß Two Ways to Work with Code in Python

This is SUPER IMPORTANT! There are two different ways to call operations in Python, and they look different:

‚≠ê Way #1: Built-in Functions (like print and input)

These are general-purpose functions that work on their own. You put the data inside the parentheses.

# Built-in Functions - data goes INSIDE parentheses
print("Hello World") # Function name, then (data)
len([1, 2, 3]) # Function name, then (data)
input("Enter name: ") # Function name, then (data)
max([5, 2, 8]) # Function name, then (data)
üìå Pattern: function_name(data)

‚≠ê Way #2: List Methods (like .append and .sort)

These are special operations that belong to lists. You start with the list name, then use a dot (.), then the method name. These methods change or work with that specific list.

# List Methods - list name FIRST, then dot, then method
my_list = [1, 2, 3]
my_list.append(4) # list_name.method(data)
my_list.sort() # list_name.method()
my_list.reverse() # list_name.method()
my_list.pop(0) # list_name.method(data)
üìå Pattern: list_name.method_name()

‚ùå Common Mistakes to Avoid

‚ùå WRONG - Trying to use list methods like functions:
append(my_list, 5) # ‚ùå This doesn't work!
sort(my_list) # ‚ùå This doesn't work!
reverse(my_list) # ‚ùå This doesn't work!
‚úÖ CORRECT - List methods need the list name first with a dot:
my_list.append(5) # ‚úÖ Correct!
my_list.sort() # ‚úÖ Correct!
my_list.reverse() # ‚úÖ Correct!
‚ùå WRONG - Trying to use functions like methods:
"Hello".print() # ‚ùå This doesn't work!
my_list.len() # ‚ùå This doesn't work!
name.input() # ‚ùå This doesn't work!
‚úÖ CORRECT - Functions take data inside parentheses:
print("Hello") # ‚úÖ Correct!
len(my_list) # ‚úÖ Correct!
input("Enter name: ") # ‚úÖ Correct!

üß† How to Remember the Difference

üîπ Built-in Functions (print, len, input):

  • Work on ANY type of data (strings, lists, numbers, etc.)
  • Stand alone - don't need a specific object
  • Think: "I want to print something" ‚Üí print(something)
  • Think: "I want the length of something" ‚Üí len(something)

üîπ List Methods (.append, .sort, .remove):

  • Only work on lists - they belong to the list object
  • Need a specific list to operate on
  • Think: "I want my list to append something" ‚Üí my_list.append(something)
  • Think: "I want this list to sort itself" ‚Üí this_list.sort()

Welcome! In this playbook, we'll use colorful shapes to understand how Python list methods work. Each shape represents an item in a list: üî¥ Red Circle, üîµ Blue Square, and üî∫ Orange Triangle.

.append()
üìå Adds an item to the END of the list. Think of it like adding a new friend to the end of a line!
Before:
my_list.append(üî∫)
After:
fruits = ["apple", "banana"]
fruits.append("orange")
print(fruits) # ["apple", "banana", "orange"]
.insert()
üìç Inserts an item at a SPECIFIC position. Like cutting in line at position 1!
Before:
my_list.insert(1, üî∫)
After:
colors = ["red", "blue", "blue"]
colors.insert(1, "yellow")
print(colors) # ["red", "yellow", "blue", "blue"]
.pop()
üéà Removes and returns an item at a specific position (or the last item if no position given).
Before:
my_list.pop(1)
After:
numbers = [10, 20, 30, 40]
removed = numbers.pop(1) # Removes 20
print(numbers) # [10, 30, 40]
print(removed) # 20
.remove()
‚ùå Removes the FIRST occurrence of a specific value from the list.
Before:
my_list.remove(üîµ)
After:
pets = ["cat", "dog", "dog", "bird"]
pets.remove("dog") # Removes first "dog"
print(pets) # ["cat", "dog", "bird"]
.reverse()
üîÑ Reverses the order of items in the list. Like reading a word backwards!
Before:
my_list.reverse()
After:
letters = ["A", "B", "C", "D"]
letters.reverse()
print(letters) # ["D", "C", "B", "A"]
.sort()
üìä Sorts the list in ascending order (smallest to largest, or A to Z).
Before:
my_list.sort()
After:
numbers = [5, 2, 8, 1, 9]
numbers.sort()
print(numbers) # [1, 2, 5, 8, 9]
.index()
üîç Returns the position (index) of the FIRST occurrence of an item.
List:
my_list.index(üî∫)
Returns: 2
(Remember: Python counts from 0! So position 2 is the 3rd item)
fruits = ["apple", "banana", "cherry"]
position = fruits.index("cherry")
print(position) # 2
.count()
üî¢ Counts how many times an item appears in the list.
List:
my_list.count(üîµ)
Returns: 2
(There are 2 blue squares in the list)
scores = [10, 20, 10, 30, 10]
how_many = scores.count(10)
print(how_many) # 3

üìä Sorts the list in ascending order (smallest to largest, or A to Z)

.index()
üîç Returns the position (index) of the FIRST occurrence of an item

.count()
üî¢ Counts how many times an item appears in the list
