What is Binary?

Computers only understand two digits: 0 and 1. This is called the binary number system (base-2).

We humans use the decimal number system (base-10), which has ten digits: 0 through 9. People have 10 fingers — that's why base-10 feels natural. Computers have only two "fingers": 0 and 1.

Number SystemBaseDigits UsedExample
DecimalBase-100, 1, 2, …, 942
BinaryBase-20, 1101010
💡 Tip: Every number you see on a screen is stored as binary inside the computer!

Place Values in Binary

In decimal, each place is a power of 10:

Hundreds   Tens   Ones
  10²      10¹    10⁰
  100       10      1

In binary, each place is a power of 2:

 128    64    32    16     8     4     2     1
  2⁷    2⁶    2⁵    2⁴    2³    2²    2¹    2⁰
💡 Key idea: Each position in a binary number represents a power of 2, just like each position in a decimal number represents a power of 10.

Method: Place Value Addition

Multiply each binary digit by its place value (power of 2) and add them all up.

Example: Convert 10010101 to decimal

Position:  2⁷    2⁶    2⁵    2⁴    2³    2²    2¹    2⁰
Value:    128    64    32    16     8     4     2     1
Bits:       1     0     0     1     0     1     0     1

= 128 + 0 + 0 + 16 + 0 + 4 + 0 + 1
= 149

Example: Convert 11111111 to decimal

= 128 + 64 + 32 + 16 + 8 + 4 + 2 + 1
= 255
💡 Key insight: An 8-bit binary number can represent values from 0 (00000000) to 255 (11111111).

Method: Successive Division by 2

Repeatedly divide by 2 and record the remainders. Read the remainders from bottom to top (last remainder is the MSB).

Example: Convert 75 to binary

75 ÷ 2 = 37  remainder 1  ← LSB (Least Significant Bit)
37 ÷ 2 = 18  remainder 1
18 ÷ 2 =  9  remainder 0
 9 ÷ 2 =  4  remainder 1
 4 ÷ 2 =  2  remainder 0
 2 ÷ 2 =  1  remainder 0
 1 ÷ 2 =  0  remainder 1  ← MSB (Most Significant Bit)

Read bottom → top: 1001011
✅ Check: 64 + 0 + 0 + 8 + 0 + 2 + 1 = 75 ✓
