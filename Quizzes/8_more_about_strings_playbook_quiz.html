Strings = Character Lists

A string in Python is a sequence of characters — think of it as a list where each element is a single character. You can even convert a string to a list using list():

strings_as_lists.py 
name = 'Siva Jasthi'
print(type(name))

name_lst = list(name)
print(type(name_lst))
print(name_lst)
💡 Each character in the string — including spaces — gets its own index position, just like elements in a list.

indexing.py 
lang = 'PYTHON'

print(lang[0])    # P  — first character
print(lang[5])    # N  — last character
print(lang[-1])   # N  — last (negative)
print(lang[-6])   # P  — first (negative)
⚠️ IndexError: Accessing lang[6] or lang[-7] raises an IndexError because those positions don't exist in a 6-character string!
✂️
String Slicing

Slicing extracts a portion of a string using the syntax: string[start : stop : step]

  • start — where the slice begins (inclusive, default 0)
  • stop — where the slice ends (exclusive, default end)
  • step — how many to skip (default 1)
slicing.py 
text = 'Programming'

print(text[1:3])     # ro
print(text[:3])      # Pro
print(text[-2:])     # ng
print(text[:])       # Programming (full copy)
print(text[1:5:2])   # rg  (step of 2)
print(text[1::2])    # rgamn (every other from index 1)
print(text[::-1])    # gnimmargorP (REVERSED!)
💡 text[::-1] is a quick and elegant way to reverse any string in Python!
🔒
Strings are Immutable

Strings cannot be changed after they are created. Trying to assign a new character to a specific index raises a TypeError:

immutable_error.py 
name = 'Python'
name[0] = 'C'   # ❌ TypeError!

To "modify" a string, you build a brand new one using slicing and concatenation:

workaround.py 
input_str = 'Python'
print('Before:', input_str)

slice1 = input_str[1:]       # 'ython'
new_string = 'C' + slice1   # 'Cython'

print('New String:', new_string)
print('Original:', input_str)
ℹ️ Python creates a new string object in memory. The original string stays unchanged — eventually the old orphan is cleaned up by garbage collection.
🔍
Membership & Traversal

Use in and not in to check if a substring exists (case-sensitive!):

membership.py 
text = 'Python programming is cool'

print('python' in text)       # False (case-sensitive!)
print('Python' in text)       # True
print('python' not in text)   # True
print('cool' in text)         # True

Traverse a string with a for loop — three different ways:

traversal.py 
name = 'Python'

# Method 1: Direct (values only)
for ch in name:
    print(ch, end=',')

print()  # newline

# Method 2: Index-based with range
for idx in range(len(name)):
    print(idx, '-->', name[idx])

# Method 3: enumerate (index + value)
for idx, value in enumerate(name):
    print(idx, value)

String Testing Methods (Querying / Asking)

These methods return True or False — great for validating user input!

MethodReturns True when…
isalnum()String has only letters or digits
isalpha()String has only alphabetic letters
isdigit()String has only numeric digits
islower()All letters are lowercase
isupper()All letters are uppercase
isspace()String has only whitespace
testing_methods.py 
s1 = '123'
print(s1.isdigit())    # True
print(s1.isalpha())    # False

s2 = 'PYTHON'
print(s2.isupper())    # True
print(s2.islower())    # False

s3 = 'Hello123'
print(s3.isalnum())    # True (letters + digits)
print(s3.isalpha())    # False (has digits)

String Modification Methods

These methods return a new string — they do NOT change the original (because strings are immutable!).

MethodWhat it does
lower()All letters → lowercase
upper()All letters → uppercase
strip()Removes leading & trailing whitespace
lstrip()Removes leading (left) whitespace
rstrip()Removes trailing (right) whitespace
title()First letter of each word → uppercase
swapcase()Swaps lower ↔ upper
modification.py 
text = 'Python Programming is COOL'

print(text.lower())       # python programming is cool
print(text.upper())       # PYTHON PROGRAMMING IS COOL
print(text.swapcase())    # pYTHON pROGRAMMING IS cool
print(text.title())       # Python Programming Is Cool

# Original is unchanged!
print(text)
⚠️ All string methods return new values. To keep the result, assign it: text = text.upper()

Search, Replace & Count
MethodWhat it does
find(sub)Index of first occurrence; -1 if not found
count(sub)How many times sub appears
replace(old, new)Replace all occurrences of old with new
startswith(sub)True if string starts with sub
endswith(sub)True if string ends with sub
search_replace.py 
title = 'Oh boy! Python programming is cool'

print(title.count('o'))                # 5
print(title.lower().count('o'))        # 6 (case-insensitive!)
print(title.find('Python'))            # 8
print(title.replace('cool', 'fun'))  # Oh boy! Python programming is fun
print(title.startswith('Oh'))        # True
print(title.endswith('cool'))         # True
💡 Method chaining: title.lower().count('o') first converts to lowercase, then counts — giving a case-insensitive count!
✂️
Splitting & Joining Strings

split() breaks a string into a list. join() glues a list back into a string.

split_join.py 
# Splitting by whitespace (default)
sentence = 'Python programming is cool'
words = sentence.split()
print(words)
print('Word count:', len(words))

# Splitting by a custom delimiter
date = '1/22/2023'
parts = date.split('/')
print(parts)
print('Year:', parts[2])

# Joining with a hyphen
result = '-'.join(words)
print(result)

Built-in Functions on Strings

Python's built-in functions like len(), max(), and min() work on strings:

builtins.py 
name = 'india'
print('Length:', len(name))    # 5
print('Max:',    max(name))    # n (highest alphabetically)
print('Min:',    min(name))    # a (lowest alphabetically)

print()
name2 = 'Indigo'
print('Max:',    max(name2))   # o
print('Min:',    min(name2))   # I (uppercase < lowercase!)
ℹ️ In ASCII/Unicode, uppercase letters (A–Z) have lower values than lowercase (a–z). So min('Indigo') returns 'I' — not 'd'!
✨
Formatted Strings (f-strings)

F-strings let you embed expressions directly inside a string using { } placeholders, prefixed with f:

fstrings.py 
name = 'Alex'
marks = 40
gpa = 3.9

# Old way (messy with commas)
print("Hi", name, "! You got", marks, "marks.")

# f-string (clean and easy!)
print(f"Hi {name}! You got {marks} marks. GPA is {gpa}")

# Controlling decimal places
import math
print(f"Pi ≈ {math.pi:.3f}")
print(f"Pi ≈ {math.pi:.5f}")
✖️
Repetition Operator ( * )

The * operator repeats a string a given number of times:

repetition.py 
star = '* '
print(star * 30)

dash = '-'
print(dash * 40)

print('Ha' * 5)
