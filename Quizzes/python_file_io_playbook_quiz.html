File I/O Concepts

1Ô∏è‚É£ read() - Read Everything at Once

What it does: Reads the ENTIRE file content as one big string.

with open('myfile.txt') as file: content = file.read() print(content) # Prints everything!
üí° When to use: When you want ALL the content at once. Like reading an entire book in one go!

2Ô∏è‚É£ read(n) - Read N Characters

What it does: Reads only a specific number of characters from the file.

with open('myfile.txt') as file: first_10 = file.read(10) # Read 10 characters next_5 = file.read(5) # Read 5 more
üí° When to use: When you want to read data in chunks. Like eating pizza slice by slice!

3Ô∏è‚É£ readline() - Read One Line at a Time

What it does: Reads ONE line from the file (up to the newline character).

with open('myfile.txt') as file: line1 = file.readline() # First line line2 = file.readline() # Second line
üí° When to use: When you want to process one line at a time. Like reading a poem verse by verse!

4Ô∏è‚É£ readlines() - Read All Lines as a List

What it does: Reads ALL lines and returns them as a list of strings.

with open('myfile.txt') as file: all_lines = file.readlines() # Returns: ['line1\n', 'line2\n', 'line3\n']
üí° When to use: When you want all lines in a list to loop through. Like getting a playlist of all songs!

5Ô∏è‚É£ with Statement - The Safe Way to Handle Files

What it does: Automatically closes the file when done (even if errors occur).

# ‚ùå Old way (not recommended) file = open('myfile.txt') content = file.read() file.close() # Must remember to close! # ‚úÖ Better way (recommended) with open('myfile.txt') as file: content = file.read() # File automatically closes here!
üí° Why it's important: The with statement is like a responsible assistant that cleans up after itself automatically. Always use it!
