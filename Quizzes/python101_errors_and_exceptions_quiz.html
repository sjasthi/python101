Question 1
Which of the following is used to handle exceptions in Python?
‚úÖ Correct Answer: C - try - except
Python uses try-except blocks for exception handling, not try-catch like Java/C++.
Question 2
What exception is raised when you try to access an undefined variable?
‚úÖ Correct Answer: A - NameError
NameError is raised when you try to use a variable that hasn't been defined yet.
Question 3
Which block is always executed, even if an exception is not raised?
‚úÖ Correct Answer: B - finally
The finally block always executes, whether an exception occurs or not.
Question 4
What would Python print?
try:
    print(1 / 0)
except ZeroDivisionError:
    print("Cannot divide by zero!")
‚úÖ Correct Answer: B - Cannot divide by zero!
The division by zero raises ZeroDivisionError, which is caught by the except block.
Question 5
What will happen if no exception occurs in a try block?
‚úÖ Correct Answer: B - The else block will run (if present).
The else block only runs when NO exception occurs in the try block.
Question 6
What would Python print?
try:
    x = "Hello"
    x = len(x)
except KeyError:
    x = 20
except IndexError:
    x = 10
finally:
    print(x)
‚úÖ Correct Answer: C - 5
No exception occurs: x = "Hello" (5 characters), x = len(x) = 5, finally prints 5.
Question 7
What would Python print?
x = 10
try:
    x = 10/0
except:
    pass

print(x)
‚úÖ Correct Answer: A - 10
x starts as 10, the division fails, except catches it but does nothing (pass), so x remains 10.
Question 8

x = 2
try:
    x = x // 2

    print(x, end='')

    print(x+1, end='')

    print(x, end='')

print(x)

Question 9
What does the raise keyword do in Python?
‚úÖ Correct Answer: D - Triggers an exception manually.
The raise keyword is used to manually trigger exceptions in Python.
Question 10
What type of error will the following code raise?
a = int("hello")
‚úÖ Correct Answer: B - ValueError
ValueError is raised when int() receives a string that can't be converted to an integer.
Question 11
Which statement about exception handling is TRUE?
‚úÖ Correct Answer: A - You can have multiple except blocks for different exceptions
Python allows multiple except blocks to handle different types of exceptions separately.
