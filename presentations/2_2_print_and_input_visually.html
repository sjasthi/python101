üì∫ print() Function

Purpose: Displays output to the screen

What it does:

  • Shows text, numbers, or variables
  • Automatically adds a new line
  • Can print multiple items at once
  • Converts everything to strings

‚å®Ô∏è input() Function

Purpose: Gets user input from keyboard

What it does:

  • Pauses program execution
  • Waits for user to type something
  • Returns what user typed as a string
  • Can show a prompt message

üéØ Key Concepts

üì∫ print() Details

  • Automatic conversion - Turns everything into text
  • Multiple arguments - Can print many values at once
  • Separators - Use sep parameter to change spacing
  • Line endings - Use end parameter to control new lines

‚å®Ô∏è input() Details

  • Always returns string - Even if user types numbers
  • Optional prompt - Show message to user
  • Type conversion - Use int(), float() to convert
  • Waits for Enter - Program pauses until user presses Enter

üîÑ Common Pattern

# 1. Ask user a question

# 2. Process the input

# 3. Show the result
print(greeting)
