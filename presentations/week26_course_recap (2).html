Chapter 1
Introduction to Python
🐍
Key Concepts
  • Python is a high-level, easy-to-read programming language
  • print() displays output to the screen
  • Comments start with # — Python ignores them
  • Code runs line by line, top to bottom
  • Python is case-sensitive: Print ≠ print
  • Indentation (4 spaces) defines code blocks
  • input() reads text typed by the user
Code Snapshot
# My first Python program
print("Hello, World!")
# Output: Hello, World!

print(2 + 3)       # Output: 5

# Ask the user for their name
name = input("Your name? ")
print("Hello,", name)

Conditionals
🔀
Key Concepts

Chapter 5
Functions
⚙️
Key Concepts
  • Define once with def, call many times
  • Parameters — inputs listed in the function header
  • Arguments — values passed when calling
  • return sends a value back to the caller
  • No return → function returns None
  • Default parameters: def greet(name="World")
  • Variables inside a function are local (not visible outside)
Code Snapshot
def greet(name):
    print(f"Hello, {name}!")

greet("Ali")     # Hello, Ali!

# Function with return value
def add(a, b):
    return a + b

result = add(3, 5)
print(result)     # 8

# Default parameter
def power(base, exp=2):
    return base ** exp

print(power(4))     # 16
print(power(2, 3))  # 8
Chapter 6.1
Files
📂
Key Concepts
  • Use open(filename, mode) to open a file
  • Mode "r" — read  ·  "w" — write  ·  "a" — append
  • Always close files — or use with open(…) as f: (best practice)
  • f.read() — returns entire file as one string
  • f.readlines() — returns a list of lines
  • f.write(text) — writes a string to the file
  • "w" overwrites the file; "a" adds to the end
Code Snapshot
# Writing to a file
with open("notes.txt", "w") as f:
    f.write("Hello!\n")
    f.write("Python is fun.\n")

# Reading a file (all at once)
with open("notes.txt", "r") as f:
    content = f.read()
    print(content)

# Reading line by line
with open("notes.txt", "r") as f:
    for line in f:
        print(line.strip())
Chapter 6.2
Exceptions
🚨
Key Concepts
  • An exception is a runtime error that crashes the program
  • try / except catches errors and handles them gracefully
  • else block runs only if no exception occurred
  • finally always runs — great for cleanup
  • raise lets you trigger your own exception
Common Exceptions
ExceptionCaused By
ValueError int("abc")
ZeroDivisionError10 / 0
FileNotFoundErrorfile doesn't exist
IndexError index out of range
KeyError missing dictionary key
Code Snapshot
# Basic try / except
try:
    num = int(input("Enter a number: "))
    print(10 / num)
except ValueError:
    print("That's not a number!")
except ZeroDivisionError:
    print("Cannot divide by zero!")
else:
    print("Calculation succeeded!")
finally:
    print("Done.")   # always runs

Chapter 7.2
Tuples
🔒
Lists vs Tuples
FeatureListTuple
Ordered ✅✅
Indexed ✅✅
Update items ✅❌
Duplicates ✅✅
Syntax [ ]( )

print(days.index("Wed"))   # 2

# ❌ Raises TypeError!
# days[0] = "Sun"

# Single-element tuple
one = (42,)                # comma required!
print(type(one))           # <class 'tuple'>
Chapter 8
Strings
🔤
Key Concepts
  • Strings are sequences of characters — indexed like lists
  • Positive index: left→right starting at 0
  • Negative index: right→left starting at −1
  • Slicing: s[start : stop : step]
  • Strings are immutable — methods return new strings

Chapter 9.1
Sets
🔵
Key Concepts
  • Unordered · No duplicates · Uses { }
  • CRUD: Create ✅ · Read (membership) ✅ · Update ✅ · Delete ✅
  • Cannot index with [ ] — no guaranteed order
  • Very fast membership check: if x in my_set
  • Great for removing duplicates from a list
Set Operations
OperationSymbolMeaning
Union | all elements from both
Intersection &elements in BOTH sets
Difference - in A but not in B
Code Snapshot
# Duplicates removed automatically!
nums = {1, 2, 3, 2, 1}
print(nums)            # {1, 2, 3}

# Update / Delete
nums.add(4)
nums.discard(2)

# Set operations
a = {1, 2, 3}
b = {2, 3, 4}
print(a | b)           # {1,2,3,4} union
print(a & b)           # {2,3}     intersection
print(a - b)           # {1}       difference

# Remove duplicates from list
lst = [1,2,2,3,3]
print(list(set(lst)))  # [1, 2, 3]

Chapter 10
Binary & Bitwise Operations
⚡
Binary Conversion
  • Computers store all data as 1s and 0s (binary)
  • Decimal → Binary: divide by 2, read remainders bottom-up
  • Binary → Decimal: multiply each bit by its place value (2ⁿ)
  • bin(13) → '0b1101'
  • int("1101", 2) → 13
Bitwise Operators
OpSymbolRule
AND & 1 only if BOTH bits are 1
OR | 1 if EITHER bit is 1
XOR ^ 1 if bits are DIFFERENT
NOT ~ flips all bits
Left Shift <<multiply by 2ⁿ
Right Shift>>divide by 2ⁿ
Code Snapshot
# Conversion
print(bin(13))           # 0b1101
print(int("1101", 2))    # 13

# Bitwise examples
a = 13    # 1 1 0 1
b = 10    # 1 0 1 0

print(a & b)      # 8   (1000) AND
print(a | b)      # 15  (1111) OR
print(a ^ b)      # 7   (0111) XOR
print(a << 1)     # 26  left shift × 2
print(a >> 1)     # 6   right shift ÷ 2

# Even/odd trick with AND
print(7 & 1)      # 1 → odd
print(8 & 1)      # 0 → even
